## 虚拟Hosts
Android 虚拟Hosts工具，通过 VPN 模式实现，无需 root。
本项目基于 https://github.com/x-falcon/Virtual-Hosts 分叉并修改。

## 修改
相对于原版Virtual Hosts，我更改了操作逻辑，
现在更改远程Hosts文件链接时不会立即下载，而是在每次打开vpn按钮时下载，以确保Hosts文件是最新状态。
去除了本地文件功能，
去除了首页的多余按钮，仅保留设置按钮

## Remote Virtual Hosts
An Android Virtual Hosts tool implemented via VPN mode, requiring no root access.
Forked and modified from the original repository at https://github.com/x-falcon/Virtual-Hosts

## Modifications
Compared to the original Virtual Hosts, I have revised the operational logic:

## The remote Hosts file link no longer triggers an immediate download. Instead, the file is downloaded each time the VPN button is activated to ensure the Hosts file remains up-to-date.
Removed local file functionality.
Eliminated redundant buttons on the homepage, retaining only the settings button.
